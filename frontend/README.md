# Frontend

This folder contains the user interface for:
- Applicants
- Residents
- Property Managers

Tech stack will be decided by the team.

-- How to run the UI --
1. Navigate to the folder: cd frontend
2. Install Dependencies: npm install
3. Launch: npm run dev (or npm start depending on setup)

-- Features --
1. Login Dashboard: Role-based navigation for Admin, Resident, and Applicant views
2. Applicant Dashboard: Interactive progress bar, Document Upload Center, and AI Placeholder
3. Admin Dashboard: Applicant review queue, unit inventory tracking, and announcments section
4. Resident Dashboard: Rent payment interface and maintenance request system
